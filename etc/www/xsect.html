Plan view:

Enter X for center point. Range is 0.0 to 1.0, 0.5 is normal.

Cross-section:

Enter X for waypoint (0.0 <= x < 1.0) OR I for waypoint in gridpoints (e.g. 50).

For latitude append with the letter "l" (e.g. 40.0l)

Sounding or All-Sky:

Enter I for sounding location in gridpoints (e.g. 50)

For latitude append with the letter "l" (e.g. 40.0l)

All-Sky:

The height above ground level can optionally be added with this syntax (40.0l,20.)
