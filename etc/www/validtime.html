Valid Time:

For an analysis or forecast, type in the desired valid time yydddhhmm. Here yy is the rightmost two digits of the year, ddd is the day or year (sometimes called Julian Day), hh is hour and mm is minute. If you leave it blank, the latest analysis time will be assumed.

For animations where the number of frames > 1 the valid time represents the end of the loop.

Note that forecast valid time (initial time plus forecast duration) is used in lieu of model initialization time. This makes it easier for the web page to time match forecasts with analyses and observations.
